仰卧哑铃推举【译】

发表于 2017-03-25

原文：https://www.bodybuilding.com/exercises/detail/view/name/dumbbell-bench-press

类型：力量

锻炼主要肌肉：胸肌

器械：哑铃

级别：入门

仰卧哑铃推举示例图

指南

  1. 平躺在水平长凳上，双手握住哑铃放在大腿上。手掌对立朝向的方位。

  2. 然后，借助大腿的帮助将哑铃同时举起，哑铃的位置应在肩的正前方并与肩同宽。

  3. 调整到肩宽后，旋转你的手腕，使手掌朝脚、手背朝脸。哑铃的位置应该刚好与你的胸对应，然后使你的上臂与前臂成90度，确保随时控制住哑铃，这将是你的起始位置。

  4. 然后，呼气，使用你的胸肌举起哑铃。举起后锁住你的手臂并挤压你的胸肌，保持1秒钟，然后缓慢放下置起始位置。

  5. 重复以上，即你训练计划中所规定的重复次数。

    ​

小心：

完成时，不要直接丢掉哑铃，这样会危险的扭伤你的肩膀或其他部位。

应该抬起你弯曲膝盖腿，转动你的手腕使手掌对立朝向，然后把哑铃放在大腿上。这样的结合动作所产生的动力将帮助你回到初始位置。这个时候可以把哑铃放到地板上了。

变体：

这种锻炼的其他变体是用手掌相对方式举哑铃。

此外，你可以使用手掌相对的方式与转动手腕关节举起哑铃。
